# Discord Welcome Bot (Railway FIXED)

## Features
- Welcome image (no canvas)
- Goodbye image
- Runs 24/24 on Railway
- No build errors

## Setup
1. Install Node.js 18
2. Run:
   npm install
3. Edit .env:
   - TOKEN
   - WELCOME_CHANNEL
4. Start:
   npm start

## Deploy Railway
- Upload to GitHub
- Connect Railway
- Add ENV variables
- Deploy
